get_text
Entity: function

set_text
Entity: predicate

Usage:
Text is get_text(Window)
set_text(Text , Window)

Gets/Sets the text value of a window. For the overlapping window this is the text on the Title Bar, for the edit control this is the text inside it, for the button it is the text on it and so on.
 
 
Text  string containing or receiving the text
 
Window  window or control handle, use _ (underscore) for default or current window
 

Example:

?-window( _, _, win_func(_), "set_text/get_text demo", 100, 100, 300, 200).

win_func(init):-
  menu(normal, _,_,mfunc(_),"Change text").

mfunc(press):-set_text("New Text", _).
