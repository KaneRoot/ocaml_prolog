assert_in
Entity: predicate

asserta_in
Entity: predicate

Usage:
assert_in(Clause)
asserta_in(Clause)

Asserts a clause both in the run-time database (like assert) and in the text file containing the running program. (asserta_in appends in the beginning)
 
Clause  the clause to append. 

Fails if Clause is not a clause.
 

Example:

?-assert_in(test(1,2)), test(X,Y), write(X), write(" "), write(Y), nl.
