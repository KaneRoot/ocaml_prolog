color_text
Entity: predicate

color_text_back
Entity: predicate

Usage:
color_text(Window, Color)
color_text_back(Window, Color)

Gets/sets the foreground/background color for text output. Use this predicate with windows and edit controls.
 
Window  window or control handle,
use _ (underscore) for default or current window. 
Color  integer containing a color, you can use rgb to create such integer. 

Example:

?-window( _, _, win_func(_), "Edit control demo", 100, 100, 200, 200).

win_func(init):-
  window_brush(_, rgb(0, 0, 255)),    % blue
  edit(E1, _, edit_func(_), "Start Value", 10, 10, 100, 50).

edit_func(init) :-
  color_text(_, rgb(0, 255, 0)),        % green
  color_text_back(_, rgb(255, 0, 0)). % red
