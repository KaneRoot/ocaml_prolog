Installing Strawberry Prolog

Download the file Strawberry Prolog 1.9.exe and execute it on your computer. This will install Strawberry Prolog, will enter registrations in the windows registry and will create a shortcut to Prolog.exe. After the installation you can delete the downloaded file or you can keep it for future installations.
 

Uninstalling Strawberry Prolog

Simply select Control Panel, Add/Remove Programs, select Strawberry Prolog 1.9 and press Add/Remove.
